# React + Vite
## configuracion

1. creacion de proyecto de react con vite
    `npm create vite`
    - nombre del proyecto
    - react
    - javascript

2. ejecucion del proyecto 
    `cd (ruta del archivo)`
    `npm install`
    `npm run dev`

3. instalar herramientas
    `npm install -D tailwindcss postcss autoprefixer`

4. inicializar con tailwindcss
    `npx tailwindcss init -p`
